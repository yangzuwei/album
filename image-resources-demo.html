🖼️ 精美图片资源使用指南

为你的相册提供丰富、高质量的图片资源

🎯 Unsplash 直接使用

直接使用 Unsplash 的图片 URL，支持动态尺寸调整：

// 示例：自然风光图片
const image = "https://images.unsplash.com/photo-1506905925346-21bda4d32df4?ixlib=rb-4.0.3&auto=format&fit=crop&w=800&q=80";

// 缩略图版本
const thumbnail = "https://images.unsplash.com/photo-1506905925346-21bda4d32df4?ixlib=rb-4.0.3&auto=format&fit=crop&w=400&q=80";

🔄 随机图片生成

使用 Unsplash Source API 获取随机图片：

// 按主题获取随机图片
https://source.unsplash.com/800x600/?nature
https://source.unsplash.com/800x600/?city
https://source.unsplash.com/800x600/?portrait
https://source.unsplash.com/800x600/?food

📦 预设图片包

使用项目中提供的精选图片数据：

// 使用 enriched-photos.js 中的数据
const photoData = enrichedPhotoData; // 32张精选图片

// 或者使用 Unsplash API 生成更多
const api = new UnsplashAPI();
const photos = api.generatePhotoData(50);
